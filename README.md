# Coding Test Practice (코딩테스트 준비용 풀이 모음)

현업(직장) 일정이 끝난 뒤, 여유 시간에만 꾸준히 연습하는 주니어 개발자 준비용 코딩테스트 풀이 레포입니다.  
코딩테스트 플랫폼별(프로그래머스/백준/LeetCode)로 분류,  
그리고 **각 날짜별 폴더**에  
- 1) 이날 푼 “정답 풀이 js 파일”
- 2) 오답이 있었던 경우 “오답회고(markdown) 파일”  
을 정리합니다.

> ⚠️ 매일 업로드/풀이를 하는 것은 아니며,  
> 평일에는 본업 프로젝트 레포를 우선적으로 관리/개발한 뒤  
> 시간이 남는 날에만 새로운 문제풀이 및 오답회고를 진행합니다.

---

## 📁 폴더 구조
```
/programmers/ 
  └── 2024-06-15/
        ├── xxx문제명.js      # 정답 풀이 소스(필수)
        └── xxx문제명_오답회고.md  # 오답을 경험한 문제에만 작성(있을 때만)
/baekjoon/
  └── 2024-06-15/
        ├── xxx문제명.py
        └── xxx문제명_오답회고.md
/leetcode/
  └── 2024-06-15/
        ├── xxx문제명.js
        └── xxx문제명_오답회고.md
README.md
...
```

---

## ✍️ 내 기록/정리 원칙

- 각 js/py 소스파일 상단에 "문제 링크, 난이도, 핵심 풀이 요약"을 주석으로 간략하게 기록  
- 오답이 있었던 경우 반드시 해당 오답의 원인/회고/다시 풀었을 때 깨달은 점을  
  날짜별 `_오답회고.md`로 추가
- 폴더/커밋, 모든 파일명은 한글/영문 지양 없음 (목적에만 충실!)
- ReadMe, 오답노트, 오답회고는 필요시 자유롭게 markdown 사용
